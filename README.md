## setUp
1. ts-nodeの導入
  - ts-nodeは、typescriptをコンパイルせずにそのまま動かせるツール
  ```
  npm install --save typescript ts-node
  ```
1. tsconfigの作成
  ```
  ./node_modules/.bin/tsc --init
  ```
1. 実行例
   ```
   npx ts-node　./PracticeA/main.ts
   ```
